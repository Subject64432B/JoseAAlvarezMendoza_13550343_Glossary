Cell Spacing

The cellspacing attribute specifies the space, in pixels, between cells.

Note: Do not confuse this with the cellpadding attribute, which specifies the space between the cell wall and the cell content.

CSS or Cascading Style Sheets

Is a style sheet language used to describe the presentation semantics (the look and formatting) of a document written in a markup language such as HTML.

Cross-browser

Cross-browser capability allows a website or web application to be properly rendered by all browsers.
